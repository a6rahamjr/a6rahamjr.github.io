# Jobin Reji Abraham Portfolio

Static portfolio website built from the supplied portfolio archive and master CV.

## Open locally

Open `index.html` in a browser. The site has no build step and no external runtime dependency.

## Contents

- `index.html` - single-page portfolio
- `css/styles.css` - responsive layout, theme, and visual system
- `js/main.js` - theme toggle, mobile navigation, scroll progress, and project filtering
- `assets/hero-ai-ml-workstation.png` - generated hero image for the portfolio
- `assets/Jobin_Reji_Abraham_CV.docx` - downloadable CV
